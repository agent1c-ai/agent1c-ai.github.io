What This Is

Agent1c.me is an autonomous-agent workspace built on HedgeyOS. Hitomi (the tiny hedgehog) runs with your chosen model and tools while the tab is open.

OpenAI Anthropic xAI Grok z.ai Ollama Local Telegram

Onboarding Flow

  1. Create a local vault passphrase (or skip for now).
  2. Add an AI provider key in AI APIs.
  3. Pick your model.
  4. Open chat and start talking with Hitomi.
  5. Optionally connect Telegram and local shell relay tools.

Why It Feels Different

This is an OS-like environment, not a single-page chat app. Windows, icons, notes, files, loops, and provider controls are all native parts of the desktop flow.

Everything is designed for normal users first: simple setup, direct controls, visible status, and no install-heavy pipeline.

Built On HedgeyOS

Agent1c.me reuses HedgeyOS foundations: window manager, themes, local storage model, and desktop shell behavior.

Both projects are by Decentricity.
